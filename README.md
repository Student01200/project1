# Web Development Practice Project 1

## Overview

This project is a set of web development exercises created for learning and practicing core backend web programming concepts using Java Servlets.

The project focuses on understanding how server-side applications work, including request handling, data processing, and integration with external services.

This is an educational project developed for practice purposes only.

---

## Features

### Task 1: Hash Generator Application
A simple web application that allows users to input text and generate hashed output.

Supported hashing algorithms:
- MD5
- SHA-256

Outputs include:
- Original input text
- Generated hash (Hex format)
- Generated hash (Base64 format)

---

### Task 2: Voting System Application
A basic web-based voting system that allows users to submit responses and view aggregated results.

Features:
- Submit votes via web form
- Store responses temporarily
- Display total results
- Reset results after retrieval

---

### Task 3: Country Information Application
A web application that retrieves and displays information about countries using external APIs.

Features:
- Country selection
- Display country details (name, currency, flag, neighbors)
- JSON data processing
- External REST API integration

---

## Technologies Used

- Java Servlets
- HTML5
- CSS3
- JavaScript
- JSON Processing
- REST API Integration
- MVC Pattern (basic implementation)
